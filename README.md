## INSTALLATION :

* `apt-get update -y`

* `apt-get upgrade -y`

* `apt install python-y`

* `git clone https://github.com/hack3r0/ip-info`

* `cd $HOME`

* `ls`

* `cd ip-info`

* `python ip.py`
